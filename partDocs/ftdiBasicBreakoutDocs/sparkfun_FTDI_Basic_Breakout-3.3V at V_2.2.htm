SparkFun FTDI Basic Breakout Board-3.3V

SparkFun FTDI Basic Breakout Board-3.3V (DEV-09873)

This is a basic breakout board for the FTDI FT232RL USB to serial IC. The pinout of this board matches the FTDI cable to work with official Arduino and cloned 3.3V Arduino boards.

Repository Contents

  • /Hardware - All Eagle design files (.brd, .sch)
  • /Production - Test bed files and production panel files

Product Versions

  • DEV-09873- 3.3V FTDI Basic Breakout.
  • DEV-09716- 5V FTDI Basic Breakout
